Dove conservare i vostri dati

Molti useranno in vecchio metodo di scrivere tutte le foto delle proprie vacanze nella fidata chiavetta USB o nel vecchio HDD. Molti che invece le conservano solo ed esclusivamente nel cloud. Ma quale di queste è la scelta migliore? Partiamo dal fatto che in tutti i casi ci sono degli svantaggi, sia nel cloud che nell’archiviazione negli HDD o USB.

Storage nelle USB, negli HDD o nelle SSD

È il metodo più antico per conservare i nostri dati, gli HDD hanno svolto un ruolo fondamentale nell’evoluzione dei computer. Vediamo i vantaggi e gli svantaggi della memorizzazione in “locale”.

Vantaggi

Prezzo abbastanza basso anche per grandi capacità; Velocità di scrittura/lettura sempre sopra 3 MB/s ; File accessibili anche offline ;

Svantaggi

I supporti di memorizzazione hanno bisogno di manutenzione costante I file in essi sono soggetti a virus di tipo ransom Gli HDD hanno una vita media di 4 anni, e si rovinano anche senza essere utilizzati o facendo cadute I file possono essere cancellati da formattazioni sbagliate In caso di rottura i costi per il recupero dati sono elevatissimi e spesso non danno risultati pari al 100%

Archiviazione su cloud

È un metodo moderno di archiviare i nostri file, dalle foto delle vacanze ai documenti di lavoro, ma moderno non vuol dire sempre privo di difetti

Vantaggi

Essendo gestiti da enti esterne i dischi non necessitano di manutenzione costante da parte dell’utente. File non attaccabili da virus di tipo ransom File protetti da password File condivisibili in modo semplice Velocità alta con la fibra Aziende mettono a disposizione una soglia di GB gratuiti*

Svantaggi

Molto spesso bisogna acquistare degli abbonamenti mensili che a lungo andare costerebbero più di un HDD della stessa capcità); File soggetti ad attacchi hacker File accessibili solo online Velocità bassa con le vecchie ADSL Nelle mani dell’azienda sbagliata i tuoi file possono essere venduti a terzi
